## 🧬 About DotCode

DotCode is built upon the solid foundation of [Aider](https://github.com/Aider-AI/aider), 
the leading open-source AI pair programming tool. We extend Aider's powerful 
edit engine and Git integration with cutting-edge capabilities:

- **Code Graph Engine** – Deep structural understanding of your codebase
- **GraphRAG Engine** – Semantic search and knowledge retrieval over code
- **Multi-Agent System** – Planner, Developer, Reviewer, and Tester working together
- **Skill Manager** – Extensible plugin system with community-driven skills
- **Model Router** – Intelligent LLM selection based on task complexity
- **Token Optimizer** – Smart context pruning and prompt caching
